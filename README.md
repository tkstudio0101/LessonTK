# 📊 교안 봇 업데이트 주요 기능 요약 (V7.3.0 릴리스)

교안 봇 V7.3.0 릴리스 안내입니다.  

---

### 1. 모듈화 아키텍처 전환 완료

* **단일 파일 → 12개 모듈 분리**: 기존 [combined_app.html] 단일 파일 구조를 `services/`, `ui/`, `utils/` 하위 12개 모듈로 완전 분해하여 유지보수성과 협업 효율을 대폭 개선했습니다.
* **로딩 순서 안정화**: `data → state → validation → modals → llm → prompts → generation → overview → editor → api → ui` 순서 기반 의존성 체인이 확정되어 초기화 오류가 완전히 차단됩니다.
* **빌드 파이프라인 정비**: `build.js`를 통한 단일 번들 생성 흐름을 유지하면서, 신규 모듈이 빌드 대상에 올바르게 포함되도록 정비했습니다.

### 2. P1 릴리스 블로커 3건 전부 해소

* **히스토리 패널 닫기 버튼 소멸 수정**: `renderHistoryUI()`가 패널 전체를 덮어쓰던 방식을 배지·목록 요소만 개별 갱신하는 방식으로 교체하여, 닫기·삭제 버튼이 사라지는 현상을 완전히 차단했습니다.
* **Mermaid CDN 미로드 수정**: [index.html]에 Mermaid CDN 스크립트 태그 및 `mermaid.initialize({ startOnLoad: false })` 초기화 구문을 추가해 다이어그램 렌더링이 정상 동작합니다.
* **accent 색상 분열 해소**: Tailwind config와 CSS 하드코딩 간 파란색/보라색 혼용 문제를 `#7c5bf5` 보라색으로 전면 통일했습니다.

### 3. P2 중기 개선 4건 완료

* **일괄 생성 `url_context` 누락 보완**: 모든 일괄 생성 호출 경로([generation.js] 3곳)에 `fetchWithUrlContextFallback` 적용을 완료해 Notion 교안 컨벤션 실시간 참조가 일괄 생성 시에도 동작합니다.
* **한글 CP949 인코딩 깨짐 완전 제거**: 전체 코드베이스에서 깨진 한글(원먯븞 등) 패턴이 0건으로 확인되었습니다.
* **이미지 최종 Fallback UI 안내 추가**: 이미지 3단계 체인(Naver → Google → AI 생성)이 모두 실패했을 때 배너 오류 → 토스트 → 콘솔 경고 순으로 사용자에게 명확히 안내합니다.
* **실행 방식 변경**: 이미지 수집 시 버그 방지 등 사유로 `실행.bat`파일을 사용해 교안봇을 구동하도록 변경

### 4. P3 장기 개선 4건 완료

* **교안 뷰어 H2 목차 패널**: 교안 보기 모드 진입 시 H2 헤딩 기반 목차 내비게이션이 자동으로 생성되어 긴 교안을 빠르게 탐색할 수 있습니다.
* **보기↔편집 전환 스크롤 복원**: 편집 모드 진입 시 스크롤 위치를 저장하고, 보기 모드로 돌아왔을 때 동일 위치를 자동 복원합니다.
* **교사 카드 헤딩 레벨 배지**: 교사 카드 패널에 `H2 / H3` 등 헤딩 레벨 배지가 표시되어 교안 구조 파악이 쉬워졌습니다.
* **불필요한 CDN 정리**: 미사용 `html2pdf.js` CDN 태그가 완전히 제거되었습니다.

# 📊 교안 봇 업데이트 주요 기능 요약 (V7.1.0)

교안 봇 고도화 및 확장 기능 현황입니다.
※유의사항: 최신 버전 작업 중 오류 다발하여 이상 기능이 많을 수 있습니다.

### 1. 프롬프트 및 교안 품질 제어
* **문서 작성 규칙 강제화**: AI가 문서 스타일을 어기거나 불필요한 번호를 매기는 현상을 방지하여 일관된 형태의 고품질 교안을 생성합니다.
* **형식 검사 및 경고**: 생성 직후 서식 오류나 처리되지 않은 태그가 남아있는지 실시간으로 검사해 사용자에게 알려줍니다.
* **자가 평가 및 보정**: 완성된 교안을 난이도, 정확성 등 핵심 지표로 자체 평가하고, 점수가 낮은 항목은 AI가 스스로 최대 2회 다시 작성하여 품질을 보강합니다.

### 2. 실사례 이미지 적용 (파이프라인)
* **다단계 이미지 검색**: 구글 검색과 사진 공유 사이트를 우선 탐색하여 실제 사례 사진을 찾아 적용하고, 이미지가 없을 때만 최후 수단으로 AI 이미지를 생성해 비현실적 이미지 반영을 차단합니다.
* **화면 출력 속도 개선**: 텍스트로 된 교안 내용을 사용자에게 가장 먼저 보여준 뒤 이미지를 순차적으로 불러오게 하여, 체감되는 로딩 속도를 극대화했습니다.

### 3. 상태 관리 및 대용량 문서 최적화
* **저장 공간 한계 극복**: 수십 장의 이미지가 포함된 무거운 교안을 편집할 때 튕기는 크래시(용량 초과) 현상을 해결해 안정적인 무제한 작업이 가능해졌습니다.
* **백업 파일 경량화**: 외부 백업 파일 생성 시 불필요한 데이터 덩어리들을 최소화하여 백업 파일 용량을 획기적으로 줄였습니다.
* **데이터 충돌 방지**: 다량의 이미지가 텍스트 청크 상에 동시에 생성 및 저장될 때 문서 데이터가 덮어씌워지거나 유실되는 버그를 차단했습니다.

### 4. 에디터 사용성(UI/UX) 성능 향상
* **대용량 문서 멈춤 현상(Freezing) 방어**: 내용이 방대한 교안 열람/스크롤 시 에디터 동작이 일시적으로 정지되는 현상을 방어해 부드러운 작업 환경을 유지합니다.
* **파일명 일괄 지정 규칙**: 사용자가 원하는 파일명 규칙을 미리 설정해 두면, 에디터 및 묶음 다운로드 단계 전반에 걸쳐 일관된 네이밍이 적용됩니다.
* **손쉬운 로컬 폴더 재연결**: 연결된 내 컴퓨터 보관 폴더의 접근 권한이 풀렸을 때, 버튼 클릭 한 번으로 간편하게 재연결을 요청 가능한 알림창을 제공합니다.

### 5. 교사용 전용 패널 추가
* **강사 메모 (Teacher's Notes)**: 교안 에디터 우측에 강사만 볼 수 있는 개별 메모 창을 제공하며, 문서 스크롤 위치에 맞춰 확인해야 할 메모의 포커싱이 자동으로 따라갑니다.
* **발표자/수강자 용도별 파일 분배 편집**: 화면에서는 하나의 문서 구조로 통합 편집하되, 실제 파일 저장 시 '강사용(주석 포함 파일)'과 '학생용(주석 제거 파일)'으로 알아서 분기되어 떨어집니다.

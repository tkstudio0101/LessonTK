# 교안 봇 (디벨로켓 교안 도우미) - V7.2.0

AI 기반 게임 기획 교안 자동 생성 도구 배포본입니다.

## 실행 방법

### 사전 준비
- 바닐라 JS + `file://` 환경에서 실행 가능합니다.

### 실행
1. `V7.2.0` 폴더 내의 `combined_app.html` 파일을 브라우저(Chrome 권장)에서 열어 즉시 사용하실 수 있습니다. 이 파일은 모든 컴포넌트와 스크립트 모듈이 하나로 병합된 단일 파일 버전입니다.

## 주요 기능 (현재 버전 기준)

- **AI 교안 생성**: Google Gemini API 기반 자동 교안 생성 (분할 생성·일괄 생성)
- **교과·차시 관리**: 커리큘럼 설계 → 단원/차시 구조 관리 + D&D 편집
- **인라인 에디팅**: 텍스트 블록 클릭 편집 + 플로팅 툴바 (문체 변환·분량 조절)
- **교사 카드**: 섹션 앵커 기반 교사용 지도 메모 패널 (AI 생성 지원)
- **내보내기**: Markdown / HTML / PDF (`window.print()`) / ZIP (폴더 동기화)
- **이미지 삽입**: Google 검색 → AI 생성 2단계 체인 (비동기 백그라운드)
- **Notion 연동**: `url_context` 기반 교안 컨벤션·예시 실시간 참조
- **학습 이해도**: 퀴즈 생성 및 렌더링 지원
- **커맨드 팔레트**: `Ctrl+K` 단축키를 통한 12개 주요 액션 검색 및 실행 기능

## 기술 스택

| 분류 | 기술 |
|------|------|
| **AI** | Google Gemini API (url_context 지원) |
| **UI 프레임워크** | 바닐라 JS + Tailwind CSS |

# 📊 교안 봇 업데이트 주요 기능 요약 (V7.2.0)

교안 봇 고도화 및 확장 기능 현황입니다.

### 1. 대규모 모듈 분해 (아키텍처 정비)(**개발 모듈 한정**)
* **단일 파일 → 7개 모듈 분할**: 기존 `api.js` / `ui.js` 두 파일에 과적된 로직을 역할에 따라 `utils/`, `services/`, `ui/` 하위 7개 전용 파일로 완전 분리하여 유지보수성과 가독성을 대폭 개선했습니다.
* **계층별 의존성 정렬**: 데이터 → 상태 → 유틸·모달 → 서비스 → UI 순서로 로딩 순서를 재배치하여 런타임 참조 오류 발생 가능성을 최소화했습니다.
* **공개 API 시그니처 동결**: 분해 과정에서 전역 함수명 및 호출 패턴을 변경 없이 유지하여 기존 연동 코드와의 호환성을 보장합니다.

### 2. 서식·품질 관리 강화
* **컨벤션 기반 서식 린트**: `utils/validation.js`에서 L1 헤딩 이모지 누락, L2 번호형 헤딩 오용, 일반 마크다운 이미지 태그 혼입 등을 자동 감지하여 오류 항목을 즉시 보고합니다.
* **분량 임계값 경고**: 교안 본문이 700줄 미만(분량 부족)이거나 1,000줄 초과(과다)일 경우 경고를 발생시켜 적정 분량 준수를 유도합니다.
* **완성도 다항목 체크**: 학습 목표(`objectives`) · 사례(`examples`) · 퀴즈(`quiz`) · 이미지(`images`) · 개요(`overview`) 항목별 존재 여부를 자동 검사해 누락 요소를 식별합니다.

### 3. 교사용 카드 및 지도 메모
* **섹션 앵커 기반 동기화**: 에디터 스크롤 위치에 맞춰 해당 섹션의 교사 카드가 자동으로 포커싱되어 강사가 현재 강의 위치와 메모를 동시에 확인할 수 있습니다.
* **AI 기반 카드 자동 생성**: 교안 콘텐츠를 분석하여 강사용 지도 메모를 AI가 직접 초안 생성합니다. "핵심 개념 요약 1문장 + 설명 방법 1문장" 2문장 구조를 기본으로 하며, 한국어 100% 원칙을 적용합니다.
* **영어 폴백 차단**: `buildAutoTeacherGuideNotes` 내 영어 문장형 가이드(`When teaching...` 등) 혼입을 코드 수준에서 차단하여 카드 언어 정책을 강제합니다.

### 4. 이미지 파이프라인 정비
* **2단계 검색 체인 확립**: 로컬 캐시(IndexedDB) 확인 → Google Custom Search 실사례 탐색 → AI 이미지 생성 순서로 단계별 체인을 정형화하여 비현실적 AI 이미지 남발을 방지합니다. Pixabay 스톡 검색은 저작권·적합성 이슈로 파이프라인에서 제거했습니다.
* **VLM 기반 이미지 검열**: 수집된 이미지를 비전 언어 모델이 자동 검수하여 썸네일 낚시·부적절 이미지를 필터링합니다.
* **API 할당량 다중 키 롤링**: Google Custom Search 429(할당량 초과) 발생 시 등록된 복수의 API 키를 순회하여 일괄 생성 중 검색 중단 없이 연속 처리합니다.

### 5. 사용성(UI/UX) 개선
* **커맨드 팔레트 도입**: `Ctrl+K` 단축키로 12개 주요 액션(생성·내보내기·문체 변환 등)을 키워드 검색으로 즉시 실행할 수 있는 팔레트를 추가하여 메뉴 탐색 비용을 줄였습니다.
* **LNB 카드 액센트 인디케이터**: 홈 교과 목록 카드에 좌측 세로 강조 바를 추가하여 현재 호버·선택된 교과를 시각적으로 명확하게 구분합니다.
* **일괄 생성 참조 파일 다중 업로드**: 일괄 생성 시 참조용 파일을 최대 100개까지 업로드할 수 있으며, 초과 시 팝업 경고를 표시합니다.

### 6. 외부 연동 및 데이터 관리
* **Notion url_context 교안 컨벤션 참조**: Gemini API의 `url_context` 기능을 활용해 Notion에 정의된 교안 컨벤션과 예시를 생성 시점에 실시간으로 참조하여 규칙 준수율을 높입니다.
* **로컬 폴더 실시간 동기화**: `saveState()` 훅에 연결된 폴더 동기화 로직이 저장 이벤트마다 자동 실행되어 교안을 `.json` / `.md` 물리 파일로 PC에 즉시 보존합니다.
* **IndexedDB 기반 무제한 히스토리**: 브라우저 로컬 스토리지 한계를 우회하는 IndexedDB를 채택하여 대용량 이미지가 포함된 교안도 히스토리 아카이브가 가능합니다.
